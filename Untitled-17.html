C# While Loop

Hurkok

A hurkok végrehajthatnak egy kódblokkot, amíg egy meghatározott feltételt el nem érnek.

A hurkok hasznosak, mert időt takarítanak meg, csökkentik a hibákat, és olvashatóbbá teszik a kódot.

C# While Loop

A whilehurok egy kódblokkon keresztül fut, amíg a megadott feltétel True:

Szintaxis

while (condition)
{
// code block to be executed
}

Az alábbi példában a ciklus kódja újra és újra lefut, amíg az (i) változó 5-nél kisebb:

Példa

int i = 0;
while (i < 5)
{
Console.WriteLine(i);
i++;
}

A Do/While hurok

A do/whilehurok a hurok egy változata while. Ez a ciklus egyszer végrehajtja a kódblokkot, mielőtt ellenőrizné, hogy a feltétel igaz-e, majd addig ismétli a ciklust, amíg a feltétel igaz.

Szintaxis

do
{
// code block to be executed
}
while (condition);

Az alábbi példa hurkot használ do/while. A ciklus mindig legalább egyszer végrehajtásra kerül, még akkor is, ha a feltétel hamis, mivel a kódblokk végrehajtása a feltétel tesztelése előtt történik:

Példa

int i = 0;
do
{
Console.WriteLine(i);
i++;
}
while (i < 5);
