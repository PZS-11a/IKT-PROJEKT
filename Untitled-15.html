C# feltételek és If nyilatkozatok

A C# támogatja a matematikából szokásos logikai feltételeket:

Kevesebb, mint: a < b
Kisebb vagy egyenlő: a <= b
Nagyobb, mint: a > b
Nagyobb vagy egyenlő, mint: a >= b
Egyenlő a == b
Nem egyenlő: a != b
Ezekkel a feltételekkel különböző döntésekhez különböző műveleteket hajthat végre.

A C# a következő feltételes utasításokkal rendelkezik:

Használja ifa végrehajtandó kódblokk meghatározására, ha a megadott feltétel igaz
Használja elsea végrehajtandó kódblokk megadására, ha ugyanaz a feltétel hamis
Használja else ifa tesztelendő új feltétel megadására, ha az első feltétel hamis
switchSzámos alternatív végrehajtandó kódblokk megadására használható

Az if nyilatkozat

Az ifutasítás segítségével adjon meg egy C# kód blokkot, amelyet végrehajtani kell, ha a feltétel True.

Szintaxis

if (condition)
{
// block of code to be executed if the condition is True
}

Az alábbi példában két értéket tesztelünk annak megállapítására, hogy a 20 nagyobb-e, mint 18. Ha a feltétel True, nyomtasson ki egy szöveget:

Példa

if (20 > 18)
{
Console.WriteLine("20 is greater than 18");
}

Változókat is tesztelhetünk:

Példa

int x = 20;
int y = 18;
if (x > y)
{
Console.WriteLine("x is greater than y");
}

Példa elmagyarázva A fenti példában két változót, x- et és y -t használunk annak tesztelésére, hogy x nagyobb-e, mint y (az >operátor segítségével). Mivel x 20, y 18, és tudjuk, hogy 20 nagyobb, mint 18, kiírjuk a képernyőre, hogy "x nagyobb, mint y".
