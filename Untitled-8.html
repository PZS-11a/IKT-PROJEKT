Adattípusok

Adattípus Méret Leírás
int 4 bájt Egész számok tárolása -2,147,483,648-tól 2,147,483,647-ig
long 8 bájt Egész számok tárolása -9,223,372,036,854,775,808-tól 9,223,372,036,854,775,807-ig
float 4 bájt Törtszámok tárolása. Elég 6-7 tizedesjegy tárolására
double 8 bájt Törtszámok tárolása. Elég 15 tizedesjegy tárolására
bool 1 bit Értékek tárolása igaz vagy hamis formában
char 2 bájt Egyetlen karakter/tördelőjelet tárol, egyszeres idézőjelek között
string 2 bájt karakterenként Karaktersorozat tárolása, dupla idézőjelek között

// int
int integerValue = 42;

// long
long longValue = 123456789012345;

// float
float floatValue = 3.14f;

// double
double doubleValue = 2.71828;

// bool
bool boolValue = true;

// char
char charValue = 'A';

// string
string stringValue = "Hello, C#";

// Kiírjuk a változók értékeit a konzolra
Console.WriteLine($"int: {integerValue}");
Console.WriteLine($"long: {longValue}");
Console.WriteLine($"float: {floatValue}");
Console.WriteLine($"double: {doubleValue}");
Console.WriteLine($"bool: {boolValue}");
Console.WriteLine($"char: {charValue}");
Console.WriteLine($"string: {stringValue}");
