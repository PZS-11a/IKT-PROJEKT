C# kapcsoló

C# Switch utasítások

Az switchutasítás segítségével válasszon ki egyet a számos végrehajtandó kódblokk közül.

Szintaxis

switch(expression)
{
case x:
// code block
break;
case y:
// code block
break;
default:
// code block
break;
}

Ez így működik:

A switchkifejezés egyszer kerül kiértékelésre
A kifejezés értékét a rendszer összehasonlítja mindegyik értékévelcase
Ha van egyezés, a kapcsolódó kódblokk végrehajtásra kerül
A breakés defaultkulcsszavakat ebben a fejezetben később ismertetjük

Az alábbi példa a hét napjának számát használja a hétköznap nevének kiszámításához:

Példa

int day = 4;
switch (day)
{
case 1:
Console.WriteLine("Monday");
break;
case 2:
Console.WriteLine("Tuesday");
break;
case 3:
Console.WriteLine("Wednesday");
break;
case 4:
Console.WriteLine("Thursday");
break;
case 5:
Console.WriteLine("Friday");
break;
case 6:
Console.WriteLine("Saturday");
break;
case 7:
Console.WriteLine("Sunday");
break;
}
// Outputs "Thursday" (day 4)

A szünet kulcsszó

Amikor a C# elér egy break kulcsszót, kitör a kapcsolóblokkból.

Ez leállítja a további kódok és esetek tesztelését a blokkon belül.

Ha megtalálják a megfelelőt, és kész a munka, eljött a szünet ideje. Nincs szükség további tesztelésre.

Az alapértelmezett kulcsszó

A defaultkulcsszó nem kötelező, és megad egy kódot, amelyet le kell futtatni, ha nincs kis- és nagybetűs egyezés:

Példa

int day = 4;
switch (day)
{
case 6:
Console.WriteLine("Today is Saturday.");
break;
case 7:
Console.WriteLine("Today is Sunday.");
break;
default:
Console.WriteLine("Looking forward to the Weekend.");
break;
}
// Outputs "Looking forward to the Weekend."
