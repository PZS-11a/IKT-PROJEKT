Bevezetés az Osztályokhoz C#-ban:

Az osztályok alapvető építőkövei a C# programozási nyelvnek, és lehetővé teszik a fejlesztők számára az objektumorientált tervezés alkalmazását. Az osztályok egyfajta sablont vagy vázat képeznek, amely definiálja az adatokat (attribútumok vagy mezők) és az azokon végrehajtandó műveleteket (metódusok).

Az osztályok célja az adatok és a hozzájuk kapcsolódó funkciók együttes kezelése, ami lehetővé teszi a kód újrafelhasználását és a programozás egy magasabb szintjének elérését. Az objektumorientált programozás alapelvei szerint az osztályok példányai az "objektumok", amelyek az adott osztály leírásában meghatározott tulajdonságokkal és viselkedéssel rendelkeznek.

Példa C# Osztályokra:

Az alábbi példa egy egyszerű autó osztályt mutat be. Az autó osztály attribútumokat (marka, modell, évjárat) és metódusokat (osztályszintű bemutatás és példányszintű adatok kiírása) tartalmaz. A példányok segítségével modellezzük két különböző autó adatait és viselkedését.

using System; // Példa osztály
class Auto
{
// Osztály attribútumai (mezői)
public string Marka;
public string Modell;
public int Evjarat;

// Osztályszintű metódus
public static void Bemutatas()
{
Console.WriteLine("Ez egy autó osztály bemutatása.");
}

// Példányszintű metódus
public void AutoAdatainakKiirasa()
{
Console.WriteLine($"Marka: {Marka}, Modell: {Modell}, Évjárat: {Evjarat}");
}
}

class Program
{
static void Main()
{
// Osztály példányosítása
Auto auto1 = new Auto();
auto1.Marka = "Toyota";
auto1.Modell = "Corolla";
auto1.Evjarat = 2022;

// Osztály példányának használata
Auto.Bemutatas(); // Osztályszintű metódus hívása

auto1.AutoAdatainakKiirasa(); // Példányszintű metódus hívása

// Másik autó példányosítása
Auto auto2 = new Auto();
auto2.Marka = "Ford";
auto2.Modell = "Focus";
auto2.Evjarat = 2021;

auto2.AutoAdatainakKiirasa();
}
}
