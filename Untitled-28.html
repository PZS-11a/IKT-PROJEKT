C# osztály tagjai

Osztály tagjai

Az osztályokon belüli mezőket és metódusokat gyakran "osztálytagoknak" nevezik:

Példa

Hozzon létre egy Carosztályt három osztálytaggal: két mezővel és egy metódussal .

// The class
class MyClass
{
// Class members
string color = "red"; // field
int maxSpeed = 200; // field
public void fullThrottle() // method
{
Console.WriteLine("The car is going as fast as it can!");
}
}

Mezők

Az előző fejezetben megtanulta, hogy az osztályon belüli változókat mezőknek nevezzük, és ezeket az osztály objektumának létrehozásával és a pont szintaxis ( ) használatával érheti el .

A következő példa létrehozza az Carosztály objektumát myObj. Ezután kiírjuk a mezők értékét color és maxSpeed:

Példa

class Car
{
string color = "red";
int maxSpeed = 200;

static void Main(string[] args)
{
Car myObj = new Car();
Console.WriteLine(myObj.color);
Console.WriteLine(myObj.maxSpeed);
}
}

A mezőket üresen is hagyhatja, és módosíthatja azokat az objektum létrehozásakor:

Példa

class Car
{
string color;
int maxSpeed;

static void Main(string[] args)
{
Car myObj = new Car();
myObj.color = "red";
myObj.maxSpeed = 200;
Console.WriteLine(myObj.color);
Console.WriteLine(myObj.maxSpeed);
}
}

Ez különösen akkor hasznos, ha egy osztályból több objektumot hoz létre:

Példa

class Car
{
string model;
string color;
int year;

static void Main(string[] args)
{
Car Ford = new Car();
Ford.model = "Mustang";
Ford.color = "red";
Ford.year = 1969;

Car Opel = new Car();
Opel.model = "Astra";
Opel.color = "white";
Opel.year = 2005;

Console.WriteLine(Ford.model);
Console.WriteLine(Opel.model);
}
}

Objektum módszerek

A C# Methods fejezetből megtanulta , hogy a metódusok bizonyos műveletek végrehajtására szolgálnak.

A metódusok általában egy osztályhoz tartoznak, és meghatározzák, hogyan viselkedik egy osztály objektuma.

Csakúgy, mint a mezők esetében, a metódusokat pont szintaxissal érheti el. Azonban vegye figyelembe, hogy a módszernek public. És ne feledje, hogy a metódus nevét használjuk, amelyet két zárójel ()és pontosvessző követ ;a metódus meghívásához (végrehajtásához):

Példa

class Car
{
string color; // field
int maxSpeed; // field
public void fullThrottle() // method
{
Console.WriteLine("The car is going as fast as it can!");
}

static void Main(string[] args)
{
Car myObj = new Car();
myObj.fullThrottle(); // Call the method
}
}

Több osztály használata

Az utolsó fejezetből ne feledjük, hogy több osztályt is használhatunk a jobb szervezés érdekében (egyet a mezőkre és metódusokra, egyet pedig a végrehajtásra). Ez ajánlott:

prog2.cs

class Car
{
public string model;
public string color;
public int year;
public void fullThrottle()
{
Console.WriteLine("The car is going as fast as it can!");
}
}

prog.cs

class Program
{
static void Main(string[] args)
{
Car Ford = new Car();
Ford.model = "Mustang";
Ford.color = "red";
Ford.year = 1969;

Car Opel = new Car();
Opel.model = "Astra";
Opel.color = "white";
Opel.year = 2005;

Console.WriteLine(Ford.model);
Console.WriteLine(Opel.model);
}
}
