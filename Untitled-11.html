C# operátorok

Üzemeltetők

Az operátorok változókkal és értékekkel kapcsolatos műveletek végrehajtására szolgálnak.

Az alábbi példában az + operátort használjuk két érték összeadására:

Példa

int x = 100 + 50;

Bár az + operátort gyakran használják két érték összeadására, mint a fenti példában, használható egy változó és egy érték, vagy egy változó és egy másik változó összeadására is:

int sum1 = 100 + 50; // 150 (100 + 50)
int sum2 = sum1 + 250; // 400 (150 + 250)
int sum3 = sum2 + sum2; // 800 (400 + 400)

Aritmetikai operátorok

Az aritmetikai operátorok általános matematikai műveletek végrehajtására szolgálnak:

Operátor Név Leírás Példa
+ Összeadás Két értéket összead x + y
- Kivonás Egy értéket kivon egy másiktól x - y
* Szorzás Két értéket megszoroz x * y
/ Osztás Egy értéket eloszt egy másikkal x / y
% Maradékos osztás Visszaadja az osztás maradékát x % y
++ Increment Növeli a változó értékét 1-gyel x++
-- Decrement Csökkenti a változó értékét 1-gyel x--
