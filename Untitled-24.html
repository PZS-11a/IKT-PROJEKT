C# módszer túlterhelés

Módszer Túlterhelés

Metódus túlterhelés esetén több metódusnak lehet ugyanaz a neve, különböző paraméterekkel:

Példa

int MyMethod(int x)
float MyMethod(float x)
double MyMethod(double x, double y)
/>

Tekintsük a következő példát, amely két különböző típusú számokat ad hozzá:

Példa

static int PlusMethodInt(int x, int y)
{
return x + y;
}

static double PlusMethodDouble(double x, double y)
{
return x + y;
}

static void Main(string[] args)
{
int myNum1 = PlusMethodInt(8, 5);
double myNum2 = PlusMethodDouble(4.3, 6.26);
Console.WriteLine("Int: " + myNum1);
Console.WriteLine("Double: " + myNum2);
}

Ahelyett, hogy két olyan módszert határoznánk meg, amelyeknek ugyanazt kell csinálniuk, jobb, ha az egyiket túlterheljük.

Az alábbi példában túlterheljük a PlusMethod metódust, hogy működjön mind int és double:

Példa

static int PlusMethod(int x, int y)
{
return x + y;
}

static double PlusMethod(double x, double y)
{
return x + y;
}

static void Main(string[] args)
{
int myNum1 = PlusMethod(8, 5);
double myNum2 = PlusMethod(4.3, 6.26);
Console.WriteLine("Int: " + myNum1);
Console.WriteLine("Double: " + myNum2);
}
