C# karakterláncok

A karakterláncok szövegek tárolására szolgálnak.

Egy stringváltozó idézőjelekkel körülvett karakterek gyűjteményét tartalmazza:

Példa

Hozzon létre egy típusú változót string, és adjon hozzá értéket:

string greeting = "Hello";

Egy karakterlánc-változó sok szót tartalmazhat, ha akarja:

Példa

string greeting2 = "Nice to meet you!";

Húr hossza

A karakterlánc a C#-ban valójában egy objektum, amely tulajdonságokat és metódusokat tartalmaz, amelyek bizonyos műveleteket hajthatnak végre a karakterláncokon. Például egy karakterlánc hosszát a következő tulajdonsággal találhatjuk meg Length:

Példa

string txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";
Console.WriteLine("The length of the txt string is: " + txt.Length);

Egyéb módszerek

Számos karakterlánc-módszer áll rendelkezésre, például ToUpper()a és ToLower(), amely a karakterlánc nagy- vagy kisbetűssé alakított másolatát adja vissza:

Példa

string txt = "Hello World";
Console.WriteLine(txt.ToUpper()); // Outputs "HELLO WORLD"
Console.WriteLine(txt.ToLower()); // Outputs "hello world"
