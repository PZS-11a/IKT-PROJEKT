Bevezetés a C# Metódusokhoz:

A C# nyelvben a metódusok (vagy függvények) olyan kódblokkok, amelyek egy adott feladatot vagy műveletet hajtanak végre. Ezek a kódblokkok elnevezhetők és újrafelhasználhatók, ami segít a kód struktúrájának és karbantarthatóságának javításában. Egy C# program jelentős része a metódusokból áll.

A metódusok alapvetően két típusra oszthatók: osztályszintű metódusok és példányszintű metódusok. Az osztályszintű metódusokat a static kulcsszóval jelölik meg, és azok az osztályhoz tartoznak, nem pedig az osztály példányához. Példányszintű metódusok pedig az osztály példányához kötődnek.

using System;

class Program
{
// Osztályszintű metódus
static void OsztalyszintuMetodus()
{
Console.WriteLine("Ez egy osztályszintű metódus.");
}

// Példányszintű metódus
void PeldanyszintuMetodus()
{
Console.WriteLine("Ez egy példányszintű metódus.");
}

// Metódus paraméterekkel
static int Osszead(int a, int b)
{
return a + b;
}

// Metódus visszatérési értékkel
static void KiirEredmeny(int eredmeny)
{
Console.WriteLine("Az eredmény: " + eredmeny);
}

static void Main()
{
// Metódusok hívása
OsztalyszintuMetodus();

Program program = new Program();
program.PeldanyszintuMetodus();

int szam1 = 5;
int szam2 = 10;

int osszeg = Osszead(szam1, szam2);
KiirEredmeny(osszeg);

} }
