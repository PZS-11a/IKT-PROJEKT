Az előző fejezetben létrehoztunk egy Program.cs nevű C# fájlt, és a következő kóddal nyomtattuk ki a "Hello World"-et a képernyőre:

Ez itt egy példa kód, amit másolhatsz:


using System;

namespace HelloWorld {

class Program {

static void Main(string[] args) {

Console.WriteLine("Hello World!");
}

}

}

Példa elmagyarázva

1. sor: using System azt jelenti, hogy használhatunk osztályokat a Systemnévtérből.

2. sor: Üres sor. A C# figyelmen kívül hagyja a szóközt. A több sor azonban olvashatóbbá teszi a kódot.

A 3. sor: namespace a kód rendszerezésére szolgál, és ez egy tároló osztályok és más névterek számára.

4. sor: A kapcsos zárójelek {}egy kódblokk elejét és végét jelzik.

5. sor: class adatok és metódusok tárolója, amely funkcionalitást biztosít a programnak. Minden C#-ban futó kódsornak egy osztályon belül kell lennie. Példánkban az osztályt Programnak neveztük el.

7. sor: Egy másik dolog, ami mindig megjelenik a C# programokban, az a Mainmetódus. A göndör zárójelben lévő kódok {}végrehajtásra kerülnek. Nem kell értened a Main előtti és utáni kulcsszavakat. Az oktatóanyag olvasása során apránként megismerheti őket.

9. sor: Console a névtér egy osztálya System, amelynek van egy WriteLine()metódusa, amelyet szöveg kiadására/nyomtatására használnak. Példánkban a „Hello World!” szöveget írja ki.

Ha kihagyja a using Systemsort, akkor írnia kell System.Console.WriteLine()a szöveg nyomtatásához/kimenetéhez.

Megjegyzés: Minden C# utasítás pontosvesszővel végződik ;.

Megjegyzés: A C# megkülönbözteti a kis- és nagybetűket: a "MyClass" és a "myclass" jelentése eltérő.

Megjegyzés: A Java-val ellentétben a C# fájl nevének nem kell egyeznie az osztály nevével, de gyakran igen (a jobb szervezés érdekében).
