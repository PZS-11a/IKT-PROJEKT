C# módszer paraméterei

Paraméterek és érvek

Az információ paraméterként átadható a metódusoknak. A paraméterek változóként működnek a metóduson belül.

Ezeket a metódus neve után, zárójelben adjuk meg. Tetszőleges számú paramétert adhat hozzá, csak vesszővel válassza el őket.

A következő példában van egy metódus, amely egy fnamestring nevű paramétert vesz fel. A metódus meghívásakor egy keresztnevet adunk át, amelyet a metóduson belül használunk a teljes név kinyomtatására:

Példa

static void MyMethod(string fname)
{
Console.WriteLine(fname + " Refsnes");
}

static void Main(string[] args)
{
MyMethod("Liam");
MyMethod("Jenny");
MyMethod("Anja");
}

// Liam Refsnes
// Jenny Refsnes
// Anja Refsnes

Több paraméter

Tetszőleges számú paramétere lehet, csak vesszővel válassza el őket:

Példa

static void MyMethod(string fname, int age)
{
Console.WriteLine(fname + " is " + age);
}

static void Main(string[] args)
{
MyMethod("Liam", 5);
MyMethod("Jenny", 8);
MyMethod("Anja", 31);
}

// Liam is 5
// Jenny is 8
// Anja is 31
