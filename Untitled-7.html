C# változók

A változók adatértékek tárolására szolgáló tárolók.

A C#-ban különböző típusú változók vannak (különböző kulcsszavakkal definiálva), például:

int- egész számokat (egész számokat) tárol tizedesjegyek nélkül, például 123 vagy -123
double- lebegőpontos számokat tárol tizedesjegyekkel, például 19,99 vagy -19,99
char- egyedi karaktereket tárol, például "a" vagy "B". A karakterek értékeit szimpla idézőjelek veszik körül
string- tárolja a szöveget, például "Hello World". A karakterláncok értékeit idézőjelek veszik körül
bool- két állapotú értékeket tárol: igaz vagy hamis

áltozók deklarálása (létrehozása).

Változó létrehozásához meg kell adnia a típust, és hozzá kell rendelnie egy értéket:

type variableName = value;

Ahol a típus egy C# típus (például intvagy string), a változónév pedig a változó neve (például x vagy név ). Az egyenlőségjelet arra használjuk, hogy értékeket rendeljünk a változóhoz.

Ha olyan változót szeretne létrehozni, amelynek szöveget kell tárolnia, nézze meg a következő példát:

Példa

Hozzon létre egy name of type nevű változót string, és rendelje hozzá a " János " értéket:

string name = "John";
Console.WriteLine(name);

Ha olyan változót szeretne létrehozni, amelynek számot kell tárolnia, nézze meg a következő példát:

Példa

Hozzon létre egy myNum nevű változót int, és rendelje hozzá a 15 értéket :

int myNum = 15;
Console.WriteLine(myNum);

Deklarálhat egy változót az érték hozzárendelése nélkül is, és később rendelheti hozzá az értéket:

Példa

int myNum;
myNum = 15;
Console.WriteLine(myNum);

Vegye figyelembe, hogy ha új értéket rendel egy meglévő változóhoz, az felülírja az előző értéket:

Példa

Módosítsa az értékét myNum20-ra:

int myNum = 15;
myNum = 20; // myNum is now 20
Console.WriteLine(myNum);

Egyéb típusok

Más típusú változók deklarálásának bemutatása:

Példa

int myNum = 5;
double myDoubleNum = 5.99D;
char myLetter = 'D';
bool myBool = true;
string myText = "Hello";
