C# megjegyzések

A megjegyzések a C# kód magyarázatára és olvashatóbbá tételére használhatók. Használható a végrehajtás megakadályozására is alternatív kód tesztelésekor.

Egysoros megjegyzések

Az egysoros megjegyzések két perjellel ( //) kezdődnek.

A sor és a sor vége közötti szöveget //figyelmen kívül hagyja a C# (nem kerül végrehajtásra).

Ez a példa egysoros megjegyzést használ a kódsor előtt:

Példa: // This is a comment
Console.WriteLine("Hello World!");

Ez a példa egysoros megjegyzést használ a kódsor végén:

Ez itt egy példa kód, amit másolhatsz: console.log('Hello, World!');

C# többsoros megjegyzések

A többsoros megjegyzések karakterrel kezdődnek /*és ezzel végződnek */.

/*A és közötti szövegeket */a C# figyelmen kívül hagyja.

Ez a példa egy többsoros megjegyzést (egy megjegyzésblokkot) használ a kód magyarázatához:

/* The code below will print the words Hello World
to the screen, and it is amazing */
Console.WriteLine("Hello World!");
