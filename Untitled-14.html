C# Booleans

C# Booleans

A programozás során nagyon gyakran olyan adattípusra lesz szüksége, amelynek csak az egyike lehet két érték közül, például:

IGEN NEM
BE KI
IGAZ HAMIS

Ehhez a C#-nak van egy booladattípusa, amely a truevagy értékeket veheti fel false.

Logikai értékek

A logikai típust a kulcsszó deklarálja , és csak a vagy boolértékeket veheti fel :truefalse

Példa

bool isCSharpFun = true;
bool isFishTasty = false;
Console.WriteLine(isCSharpFun); // Outputs True
Console.WriteLine(isFishTasty); // Outputs False

Azonban gyakoribb, hogy logikai értékeket adnak vissza logikai kifejezésekből feltételes teszteléshez (lásd alább).

Logikai kifejezés

A logikai kifejezés logikai értéket ad vissza: Truevagy False, az értékek/változók összehasonlításával.

Ez hasznos a logika felépítéséhez és a válaszok megtalálásához.

Használhat például egy összehasonlító operátort , például a nagyobb, mint ( >) operátort, hogy megtudja, egy kifejezés (vagy változó) igaz-e:

Példa

int x = 10;
int y = 9;
Console.WriteLine(x > y); // returns True, because 10 is higher than 9

Vagy még egyszerűbb:

Példa

Console.WriteLine(10 > 9); // returns True, because 10 is higher than 9
