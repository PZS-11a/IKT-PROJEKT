C# For Loop

Ha pontosan tudja, hányszor szeretne egy kódblokkon keresztül ismételni, használja a forciklust ciklus helyett while:

Szintaxis

for (statement 1; statement 2; statement 3)
{
// code block to be executed
}

Az 1. utasítás végrehajtásra kerül (egyszer) a kódblokk végrehajtása előtt.

A 2. utasítás határozza meg a kódblokk végrehajtásának feltételét.

A 3. utasítás végrehajtásra kerül (minden alkalommal) a kódblokk végrehajtása után.

Az alábbi példa a 0-tól 4-ig terjedő számokat írja ki:

Példa

for (int i = 0; i < 5; i++)
{
Console.WriteLine(i);
}

Példa elmagyarázva

Az 1. utasítás beállít egy változót a ciklus megkezdése előtt ( int i = 0).

A 2. utasítás határozza meg a ciklus futásának feltételét ( ikisebbnek kell lennie, mint 5). Ha a feltétel true, akkor a ciklus elölről kezdődik, ha igen false, akkor a ciklus véget ér.

A 3. utasítás egy értéket ( ) növel i++minden alkalommal, amikor a ciklus kódblokkja végrehajtódik.

Egy másik példa

Ez a példa csak 0 és 10 közötti páros értékeket nyomtat:

Példa

for (int i = 0; i <= 10; i = i + 2)
{
Console.WriteLine(i);
}

Beágyazott hurkok

Lehetőség van egy hurok elhelyezésére egy másik hurok belsejében is. Ezt beágyazott huroknak nevezik .

A "belső ciklus" egyszer kerül végrehajtásra a "külső ciklus" minden egyes iterációjához:

Példa

// Outer loop
for (int i = 1; i <= 2; ++i)
{
Console.WriteLine("Outer: " + i); // Executes 2 times

// Inner loop
for (int j = 1; j <= 3; j++)
{
Console.WriteLine(" Inner: " + j); // Executes 6 times (2 * 3)
}
}
