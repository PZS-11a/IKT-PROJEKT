C# Break and Continue

C# Break

breakAz oktatóanyag egy korábbi fejezetében használt kijelentést már láthatta . Arra használták, hogy "kiugorjanak" egy switchkijelentésből.

Az breakutasítás arra is használható, hogy kiugorjon a hurokból .

Ez a példa kiugrik a ciklusból, ha iegyenlő 4:

Példa

for (int i = 0; i < 10; i++)
{
if (i == 4)
{
break;
}
Console.WriteLine(i);
}

C# Folytatás

Az continueutasítás megszakít egy iterációt (a ciklusban), ha egy meghatározott feltétel előfordul, és a ciklus következő iterációjával folytatódik.

Ez a példa kihagyja a következő értékét 4:

Példa

for (int i = 0; i < 10; i++)
{
if (i == 4)
{
continue;
}
Console.WriteLine(i);
}
