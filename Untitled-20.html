C# tömbök

Hozzon létre egy tömböt

A tömbök több értéket tárolnak egyetlen változóban, ahelyett, hogy minden egyes értékhez külön változót deklarálnának.

Egy tömb deklarálásához adja meg a változó típusát szögletes zárójelben :

string[] cars;

Most deklaráltunk egy változót, amely karakterláncok tömbjét tartalmazza.

Értékek beszúrásához használhatunk egy tömbliterált - az értékeket vesszővel elválasztott listába helyezzük, kapcsos zárójelek közé:

string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};

Egész számokból álló tömb létrehozásához a következőket írhatja:

int[] myNum = {10, 20, 30, 40};

Hozzáférés egy tömb elemeihez

Egy tömbelemet az indexszámra hivatkozva érhet el.

Ez az utasítás hozzáfér az első elem értékéhez az autókban :

Példa

string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
Console.WriteLine(cars[0]);
// Outputs Volvo

Tömbelem módosítása

Egy adott elem értékének megváltoztatásához tekintse meg az indexszámot:

Példa

cars[0] = "Opel";

Példa

string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
cars[0] = "Opel";
Console.WriteLine(cars[0]);
// Now outputs Opel instead of Volvo

Array Length

Ha meg szeretné tudni, hogy egy tömb hány elemet tartalmaz, használja a Lengthtulajdonságot:

Példa

string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};
Console.WriteLine(cars.Length);
// Outputs 4

A tömb létrehozásának egyéb módjai

Ha ismeri a C#-t, előfordulhat, hogy látott már a kulcsszóval létrehozott tömböket new, és talán látott meghatározott méretű tömböket is. A C#-ban többféleképpen hozhat létre tömböt:

// Create an array of four elements, and add values later
string[] cars = new string[4];

// Create an array of four elements and add values right away
string[] cars = new string[4] {"Volvo", "BMW", "Ford", "Mazda"};

// Create an array of four elements without specifying the size
string[] cars = new string[] {"Volvo", "BMW", "Ford", "Mazda"};

// Create an array of four elements, omitting the new keyword, and without specifying the size
string[] cars = {"Volvo", "BMW", "Ford", "Mazda"};

Rajtad múlik, hogy melyik opciót választod. Oktatóanyagunkban gyakran az utolsó lehetőséget fogjuk használni, mivel gyorsabb és könnyebben olvasható.

Megjegyzendő azonban, hogy ha deklarál egy tömböt, és később inicializálja, akkor a newkulcsszót kell használnia:

// Declare an array
string[] cars;

// Add values, using new
cars = new string[] {"Volvo", "BMW", "Ford"};

// Add values without using new (this will cause an error)
cars = {"Volvo", "BMW", "Ford"};// Declare an array
string[] cars;
