C# felhasználói bevitel

Felhasználói bevitel lekérése

Azt már megtanulta, hogy Console.WriteLine()az értékek kiadására (nyomtatására) használják. Most használni fogjuk Console.ReadLine()a felhasználói bevitelhez.

A következő példában a felhasználó megadhatja a felhasználónevét, amely a változóban van tárolva userName. Ezután kiírjuk a következő értékét userName:

// Type your username and press enter
Console.WriteLine("Enter username:");

// Create a string variable and get user input from the keyboard and store it in the variable
string userName = Console.ReadLine();

// Print the value of the variable (userName), which will display the input value
Console.WriteLine("Username is: " + userName);

Felhasználói bevitel és számok

A Console.ReadLine()metódus egy string. Ezért nem tud információt szerezni más adattípusból, például int. A következő program hibát okoz:

Hibás:

Console.WriteLine("Enter your age:");
int age = Console.ReadLine();
Console.WriteLine("Your age is: " + age);

Ahogy a hibaüzenet mondja, a „string” típust nem lehet implicit módon „int”-re konvertálni.

Szerencsére az előző fejezetből (Típuscastolás) tanulta meg , hogy bármilyen típust explicit módon konvertálhat az alábbi módszerek valamelyikével Convert.To:

Console.WriteLine("Enter your age:");
int age = Convert.ToInt32(Console.ReadLine());
Console.WriteLine("Your age is: " + age);
