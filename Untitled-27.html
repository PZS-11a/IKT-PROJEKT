C# osztályok és objektumok

Osztályok és tárgyak

Az előző fejezetből megtanulta, hogy a C# egy objektum-orientált programozási nyelv.

A C#-ban minden osztályokhoz és objektumokhoz van társítva, attribútumaival és metódusaival együtt. Például: a való életben az autó egy tárgy. Az autónak vannak olyan tulajdonságai , mint a súly és a szín, valamint olyan módszerek , mint a hajtás és a fék.

Az osztály olyan, mint egy objektumkonstruktor, vagy egy "tervrajz" az objektumok létrehozásához.

Hozzon létre egy osztályt

Osztály létrehozásához használja a classkulcsszót:

Hozzon létre egy " " nevű osztályt Caregy változóval color:

class Car
{
string color = "red";
}

Hozzon létre egy objektumot

Egy objektum jön létre egy osztályból. A nevű osztályt már létrehoztuk Car, így most ezt használhatjuk objektumok létrehozására.

Objektum létrehozásához Caradja meg az osztály nevét, majd az objektum nevét, és használja a kulcsszót new:

Példa

Hozzon létre egy " " nevű objektumot myObj, és használja ki a következő értékének kinyomtatására color:

class Car
{
string color = "red";

static void Main(string[] args)
{
Car myObj = new Car();
Console.WriteLine(myObj.color);
}
}
