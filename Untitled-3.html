C# IDE

IDE KATTINTVA LETÖLTHETED A VISUAL STUDIOT!

A C# használatának legegyszerűbb módja egy IDE használata.

A kód szerkesztésére és fordítására IDE (Integrated Development Environment) használható.

Oktatóanyagunkban a Visual Studio Communityt használjuk, amely ingyenesen letölthető a https://visualstudio.microsoft.com/vs/community/ oldalról .

A C# nyelven írt alkalmazások a .NET-keretrendszert használják, ezért célszerű a Visual Studio használata, mivel a programot, a keretrendszert és a nyelvet mind a Microsoft készítette.
